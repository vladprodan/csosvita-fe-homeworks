## Заняття 20: Патерни та оптимізація
### ☕ ДЗ #20: Архітектурні покращення

**Опис:** Реалізуйте Observer pattern для оновлення UI при зміні кошика, Module pattern для організації коду, Singleton для управління станом додатку.

**Корисні посилання:**
- [JavaScript Design Patterns](https://addyosmani.com/resources/essentialjsdesignpatterns/book/)
- [Performance Optimization](https://developer.mozilla.org/uk/docs/Web/Performance)
- [Memory Management](https://developer.mozilla.org/uk/docs/Web/JavaScript/Memory_Management)
