## Заняття 21: WebWorkers та збирання проектів
### ☕ ДЗ #21: Оптимізація та збірка проекту

**Опис:** Налаштуйте Vite для збирання проекту, винесіть важкі обчислення (фільтрація великих списків товарів) в Web Workers, оптимізуйте зображення, додайте Service Worker для кешування.

**Корисні посилання:**
- [Web Workers](https://developer.mozilla.org/uk/docs/Web/API/Web_Workers_API)
- [Vite Build Tool](https://vitejs.dev/guide/)
- [Service Workers](https://developer.mozilla.org/uk/docs/Web/API/Service_Worker_API)
